Unifi Controller config
==========

Config for [`ryansch/docker-unifi-rpi`](https://github.com/ryansch/docker-unifi-rpi/).


Updating
----------

Update the docker tag versions in [`.env`](./.env).

 * https://hub.docker.com/r/ryansch/unifi-rpi/tags

```
docker compose pull
docker compose up -d
```
